# Mini ATM System

## Project Description

The Mini ATM System is a Python-based command-line application that simulates basic ATM operations. Users can securely log in using a PIN and perform actions such as checking their balance, depositing money, withdrawing funds, viewing transaction history, and changing their PIN.

## Features

* Secure PIN authentication
* Check account balance
* Deposit money
* Withdraw money
* View mini statement
* Change PIN
* Account lock after 3 failed login attempts

## Technologies Used

* Python 3

## How to Run

```bash
python atm.py
```

## Menu Options

1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Mini Statement
5. Change PIN
6. Exit
